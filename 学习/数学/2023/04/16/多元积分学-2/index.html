二重积分的计算

1. 概念

二重积分的几何意义是将一个二元函数在一个有限的平面区域上的面积进行累加。

具体来说，如果 f(x,y)f(x,y)f(x,y) 是一个定义在区域 DDD 上的连续函数，那么 ∬Df(x,y)dA\iint_D f(x,y) dA∬D​f(x,y)dA 可以看作是在区域 DDD 中面积元素 dAdAdA 上 f(x,y)f(x,y)f(x,y) 的值的加权平均，其中 dAdAdA 表示平面上的微元面积，可以表示为 dA=dx dydA = dx\,dydA=dxdy 或 dA=dy dxdA = dy\,dxdA=dydx。

可以这么理解，二重积分和一重积分一样，基本单位都是 线 。在一重积分中，线的集合是 面 ，在二重积分中，因为多了一个维度，线的集合形成了 体 。所以求二重积分其实就是求一个体的体积，而方法是我们先 把体切成无数的面，接着用一重积分累加线的长度的方法，分别求出所有面的面积，最后 把这些面的面积累加起来就是体的体积 。

2. 步骤

计算二重积分的一般步骤如下：

  1. 确定积分区域 DDD，也就是我们要求的体是什么，可以通过画草图来确定。

  2. 写出积分被积函数 f(x,y)f(x,y)f(x,y) ，z=f(x,y)z = f(x,y)z=f(x,y) ，这个意味着在 xoyxoyxoy 平面上的每一个点 (x0,y0)(x_0,y_0)(x0​,y0​) 都有对应的一条垂直的线（平行于 zzz 轴）这条线的高度是 f(x0,y0)f(x_0,y_0)f(x0​,y0​) 。

  3. 确定积分顺序，即确定先对哪个变量积分，后对哪个变量积分。说人话就是选择一个把体切成片的方法，你切出来的片，是平行于 平面xozxozxoz 还是 平面yozyozyoz 。

  4. 进行积分计算，根据积分顺序进行分步积分。
    对于先积分 xxx 后积分 yyy 的情况，有

    ∬Df(x,y)dA=∫y=ay=bdy∫x=g1(y)x=g2(y)f(x,y)dx\iint_D f(x,y) dA = \int_{y=a}^{y=b} dy \int_{x=g_1(y)}^{x=g_2(y)} f(x,y) dx ∬D​f(x,y)dA=∫y=ay=b​dy∫x=g1​(y)x=g2​(y)​f(x,y)dx

    其中，假设我们从 xxx 轴向右，yyy 轴向上的角度看去，DDD 的上下边界为 y=ay=ay=a 和 y=by=by=b，左右边界为 x=g1(y)x=g_1(y)x=g1​(y) 和 x=g2(y)x=g_2(y)x=g2​(y)。

    而对于先积分 yyy 后积分 xxx 的情况，有

    ∬Df(x,y)dA=∫x=cx=ddx∫y=g1(x)y=g2(x)f(x,y)dy\iint_D f(x,y) dA = \int_{x=c}^{x=d}dx \int_{y=g_1(x)}^{y=g_2(x)} f(x,y) dy ∬D​f(x,y)dA=∫x=cx=d​dx∫y=g1​(x)y=g2​(x)​f(x,y)dy

    其中，假设我们从 xxx 轴向右，yyy 轴向上的角度看去，DDD 的左右边界为 x=cx=cx=c 和 x=dx=dx=d，上下边界为 y=g1(x)y=g_1(x)y=g1​(x) 和 y=g2(x)y=g_2(x)y=g2​(x)。

    需要注意的是，在具体计算时，可能需要对积分区域进行变量替换或者分割成多个子区域来简化计算。

三重积分的计算

1. 概念

三重积分的几何意义是将一个三元函数在一个有限的空间区域上的体积进行累加。

具体来说，如果 f(x,y,z)f(x,y,z)f(x,y,z) 是一个定义在空间区域 EEE 上的连续函数，那么 ∭Ef(x,y,z)dV\iiint_E f(x,y,z) dV∭E​f(x,y,z)dV 可以看作是在区域 EEE 中体积元素 dVdVdV 上 f(x,y,z)f(x,y,z)f(x,y,z) 的值的加权平均，其中 dVdVdV 表示空间中的微元体积，可以表示为 dV=dx,dy,dzdV = dx,dy,dzdV=dx,dy,dz 或 dV=dy,dx,dzdV = dy,dx,dzdV=dy,dx,dz 或 dV=dz,dx,dydV = dz,dx,dydV=dz,dx,dy。

要理解三重积分，首先要明白，三重积分就是二重积分多积分了一次。但是二重积分已经能求出几何体的体积了，三重积分的意义是什么呢？考虑一个场景，你求解一个现实世界中一个球体的质量；很明显，我们无法保证这个球体的质量是完全平均分摊在某个点的。当构成这个球体的每个点的质量可以用一个函数 f(x,y,z)f(x,y,z)f(x,y,z) 来计算的时候，我们就能通过在二重积分计算的开头，针对每个点的不同质量， “提前“ 多积分一次；这样算出来的线的 ”长度“ 就不再是长度，而是线的 ”质量“。有质量的线的集合也会是有质量的面，再进一步就是求出体的质量。

2. 步骤

计算三重积分的一般步骤如下：

  1. 确定积分区域 EEE，可以通过画图来确定。

  2. 写出积分被积函数 f(x,y,z)f(x,y,z)f(x,y,z)。

  3. 确定积分顺序，即确定先对哪个变量积分，后对哪个变量积分。可以通过画图来确定。

  4. 进行积分计算，根据积分顺序进行分步积分。具体来说，对于先积分 zzz 后积分 yyy 后积分 xxx 的情况，有

    ∭Ef(x,y,z)dV=∫x=kx=ldx∫y=g3(x)y=g4(x)dy∫z=g1(x,y)z=g2(x,y)f(x,y,z)dz\iiint_E f(x,y,z) dV = \int_{x=k}^{x=l} dx \int_{y=g_3(x)}^{y=g_4(x)} dy \int_{z=g_1(x,y)}^{z=g_2(x,y)} f(x,y,z) dz ∭E​f(x,y,z)dV=∫x=kx=l​dx∫y=g3​(x)y=g4​(x)​dy∫z=g1​(x,y)z=g2​(x,y)​f(x,y,z)dz

    其中，假设我们从 xxx 轴向右，zzz 轴向上的角度看去，EEE 的左右边界为 x=kx=kx=k 和 x=lx=lx=l ，前后边界为 y=g3(x)y=g_3(x)y=g3​(x) 和 y=g4(x)y=g_4(x)y=g4​(x) ，上下边界为 z=g1(y,z)z=g_1(y,z)z=g1​(y,z) 和 z=g2(x,y)z=g_2(x,y)z=g2​(x,y) 。

    而对于其他积分顺序，可以进行类似的计算。

    需要注意的是，在具体计算时，可能需要对积分区域进行变量替换或者分割成多个子区域来简化计算。另外，三重积分的计算通常比较繁琐，需要较高的数学功底和耐心。
